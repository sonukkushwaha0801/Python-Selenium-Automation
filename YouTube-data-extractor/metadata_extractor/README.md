# YouTube Metadata Extractor

## 🚀 Overview

A Python-based automation tool that extracts detailed metadata from a YouTube video using Selenium.
It dynamically interacts with the webpage to collect real-time information.

## 🧠 Features

* Accepts YouTube video URL as input
* Extracts video title
* Extracts channel name
* Extracts subscriber count
* Extracts views and upload time
* Extracts likes (if available)
* Saves extracted data to a text file

## 🛠 Tech Stack

* Python
* Selenium

## ⚙️ How It Works

The script launches a Chrome browser using Selenium, loads the YouTube video page, and extracts metadata by interacting with dynamic DOM elements using XPath selectors.

## ▶️ Run the Project

```bash id="plx09f"
python main.py
```

## 🎯 Purpose

Built to demonstrate real-world browser automation and dynamic data extraction from JavaScript-rendered web applications.
