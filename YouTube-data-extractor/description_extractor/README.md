# YouTube Description Extractor

## 🚀 Overview

A Python-based tool that extracts the title and description of a YouTube video using HTTP requests and HTML parsing.

## 🧠 Features

* Accepts YouTube video URL as input
* Extracts video title
* Extracts full video description using regex
* Saves description to a text file

## 🛠 Tech Stack

* Python
* Requests
* BeautifulSoup
* Regex

## ⚙️ How It Works

The script sends an HTTP request to the YouTube video page, parses the HTML content, and extracts the description using pattern matching.

## ▶️ Run the Project

```bash id="exx1rf"
python main.py
```
