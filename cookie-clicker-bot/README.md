# Cookie Clicker Bot

## 🚀 Overview

This project automates interactions with a browser-based application using Selenium, including continuous clicking and real-time upgrade decisions based on available resources.

## 🧠 Features

* Automated clicking of dynamic web elements
* Real-time data extraction from the DOM
* Conditional decision-making for upgrades
* Continuous execution loop

## 🛠 Tech Stack

* Python
* Selenium

## ⚙️ How It Works

The bot continuously clicks the main element while monitoring resource values.
When sufficient resources are available, it automatically purchases upgrades to optimize performance.

## ▶️ Run the Project

```bash
python main.py
```

* Hands-on experience with Selenium automation
* Understanding of DOM interaction and dynamic elements
* Implementation of decision-making logic
